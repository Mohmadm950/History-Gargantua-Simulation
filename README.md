🕳️ Gargantua – Black Hole Simulation

A real-time simulation of the black hole “Gargantua”, inspired by Interstellar.

Built with Java, LWJGL, and OpenGL shaders, this project focuses on real-time GPU rendering and physically inspired visual effects.

🚀 Overview

This project simulates a visually realistic black hole using custom GLSL shaders.
It is continuously evolving and being improved in terms of performance, stability, and visual fidelity.

Some parts of the development process are supported and accelerated using AI tools, mainly for experimentation, optimization, and shader development.

✨ Features
Shader-based real-time rendering (GLSL)
Accretion disk simulation with volumetric effects
Gravitational lensing approximation
Starfield distortion around the black hole
Interactive camera controls (mouse + zoom)
Ongoing performance and visual improvements
🔧 Tech Stack
Java
LWJGL (OpenGL bindings)
GLSL (custom fragment & vertex shaders)
📈 Project Status

This project is actively developed and not considered final.
Expect frequent updates, optimizations, and visual improvements.

⚠️ Requirements
Java 17+
OpenGL 3.3 compatible GPU
LWJGL dependencies properly configured (Gradle recommended)
